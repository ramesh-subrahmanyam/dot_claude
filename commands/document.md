description: Generate comprehensive documentation for the codebase
---

Read through the codebase and generate comprehensive documentation describing the design, file functionality, and overall project structure.


## Output

Generates a `docs/codebase_structure.md` file in the curret project directory containing:

- **Design Overview** - High-level architecture and design patterns
- **File Descriptions** - For each file:
  - Functionality description
  - Role in the overall project
  - Key components and responsibilities

## Documentation Goals

The generated documentation enables:

1. **Efficient Understanding** - Quickly grasp the codebase when revisiting
2. **Effective Work** - Have the context needed to make changes confidently
3. **Balanced Detail** - Not too verbose, but includes all necessary information

## Usage

Invoke this skill when you need to:
- Understand a new codebase
- Create onboarding documentation
- Generate architecture overview
- Document existing code structure
